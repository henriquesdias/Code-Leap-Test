## How to Run
 Follow the instructions to run the app locally

1. Clone this repository
2. Install all dependencies
```bash
   npm i
```   
3. Run script
```bash
  npm run dev
```
4. Access http://localhost:5173

## Build
1. Run script
```bash
npm run build
```
2. Run script
```bash
npm run preview
```
3. Access http://localhost:4173
